============================================================================== VISION AGENTS - COMPLETE SETUP GUIDE ============================================================================== PREREQUISITES: -------------- 1. Install Python 3.12.11 2. Install uv package manager: pip install uv 3. Install Git STEP 1: INSTALL DEPENDENCIES --------------------------- Run these commands in Vision-Agents folder: uv venv --python 3.12.11 uv sync --all-extras --dev pre-commit install STEP 2: SETUP ENVIRONMENT VARIABLES -------------------------------------- 1. Copy the example env file: copy .env.example .env 2. Edit .env and add your API keys. You need: REQUIRED: - STREAM_API_KEY=your_stream_api_key_here (Get from https://getstream.io/) - STREAM_API_SECRET=your_stream_api_secret_here OPTIONAL: - OPENAI_API_KEY=your_openai_api_key_here - GOOGLE_API_KEY=your_google_api_key_here - DEEPGRAM_API_KEY=your_deepgram_api_key_here - ELEVENLABS_API_KEY=your_elevenlabs_api_key_here - ANTHROPIC_API_KEY=your_anthropic_api_key_here - CARTESIA_API_KEY=your_cartesia_api_key_here - ROBOFLOW_API_KEY=your_roboflow_api_key_here STEP 3: RUN AN EXAMPLE ---------------------- Basic run: uv run examples/01_simple_agent_example/simple_agent_example.py run With video file: uv run examples/01_simple_agent_example/simple_agent_example.py run --video-track-override path\\to\\video.mp4 As HTTP server: uv run examples/01_simple_agent_example/simple_agent_example.py serve --host=0.0.0.0 --port=8000 OTHER USEFUL COMMANDS ----------------------- Run tests: uv run py.test -m not integration -n auto Code check: uv run python dev.py check Format code: uv run ruff check --fix Type checking: uv run mypy -p vision_agents
